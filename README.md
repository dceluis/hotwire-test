# README

# Install dependencies
```
bundle install
yarn install
```

# setup database
```
bin/rails db:create
bin/rails db:migrate
```

# run the server
```
bin/rails s
```

# To run specs:
```
bundle exec rspec
```

and then navigate to localhost:3000
